My Hobby

My hobby is running in my free time. Because
running helps build strong bones because it's
a weight-bearing exercise. Strengthen muscles
. Improve cardiovascular health. Burns a lot of
kilojoules.

Playing basketball can help improve motor
coordination, flexibility and stamina. It also
encourages speed, agility and strength.
Moreover, playing basketball can enhance
the function of the heart, improve the
function of the digestive system, improve
various physical fitness of the human body,
control weight and change body shape and
so on.

Tennis is an excellent cardiovascular
exercise that improves muscular strength,
endurance, balance, coordination and agility.
This also allows for increased social
interaction, since you'll need a buddy to play
the game with. Plus, tennis is a sport that can
be played at any age.
